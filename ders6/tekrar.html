CSS Tipleri

inline-internal-external

inline satır içi style attribute yazılan sadece yazılan etiket etkilenir css o etikete özgüdür.

internal aynı html dosyasının içinde head etiketleri arasında style etiketinin içerisinde yazılan css tir.

external ayrı bir css dosyası oluşturup head etiketleri arasına link ile çağrılan css tir.
